- **Dynamic Product Display:** Renders product items dynamically from a JavaScript array, updating the display in real-time.
- **Search Functionality:** Allows users to search for products by title using a search input field.
- **Sorting Options:** Provides options to sort products by price (low to high, high to low), name (A to Z, Z to A), and category.
- **Product Deletion:** Enables users to delete products by entering their ID.
- **Random Product Addition:** Allows users to add a randomly generated product to the product list.
- **Availability Status:** Displays whether a product is in stock or out of stock.
- **Responsive Design:** The application is designed to be responsive and adapt to different screen sizes.
